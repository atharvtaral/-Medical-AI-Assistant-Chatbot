# 🏥 Medical AI Assistant

A production-ready Medical Consultation Chatbot built using **OpenAI GPT-4o-mini**, **Gradio**, and **Python**.

## 🚀 Features
*   **Role-Based Interaction:** Acts as a professional Medical Consultant.
*   **Contextual Memory:** Maintains chat history for natural conversations.
*   **Professional UI:** Built with Gradio for a clean, accessible web interface.
*   **Prescription Generation:** Provides structured advice, precautions, and general medication.

## 🛠️ Tech Stack
*   **LLM:** OpenAI GPT-4o-mini
*   **Interface:** Gradio 6.0+
*   **Logic:** Python 3.10.0
*   **Environment Management:** Dotenv & Venv

## ⚙️ Setup
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add: `OPENAI_API_KEY=your_key_here`
4. Run the app: `python chatbot.py`
